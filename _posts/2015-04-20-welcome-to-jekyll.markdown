# Markdown
쉽고 멋지게 글쓰기

### 마크다운이란 무엇일까요?

일반 텍스트로 **서식이 있는 문서**를 작성하는 방법입니다.

### Markdown문법

1.Header
#Header1
##Header2
###Header3
# Header1
## Header2
### Header3
#,##,###, 등으로 으로 제목(header)작성



2.Italic
*Italic *
_  Italic  _ 
**,__으로 기울임체(italic)작성

3.Bold
** Bold **
__ Bold __ 
****,____으로 강조체(*Bold*)작성

4.Strikethrough

~ Strikethrough ~
~~으로 취소선(~Strikethrough~) 작성


5.Unordered List
-Unordered List
*Unordered List
-,*로 순서없는 리스트(unordered list) 완성

6.Ordered List

... Ordered List

...으로 순서 리스트(ordered list)작성

7.Code
` print("hi")`

`` 으로 코드(code)작성

8.Code Block
```
def power(x: int):
return x ** 2
```

`6개로 코드 블록(code block) 작성
